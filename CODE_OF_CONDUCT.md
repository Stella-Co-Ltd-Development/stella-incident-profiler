# Code of Conduct

## Our pledge

We want this project to be a respectful, inclusive, and productive open-source community.

Participants are expected to communicate professionally, assume good intent, and focus on improving the project.

## Expected behavior

- Be respectful and constructive.
- Welcome contributors with different experience levels.
- Provide actionable feedback.
- Keep discussions focused on the project.
- Respect privacy and security boundaries.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Sharing secrets, customer data, production logs, or private incident details.
- Repeatedly disrupting project discussions.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior violates this code of conduct.
